## Instalation
Update Termux : `pkg upgrade -y && pkg install bash -y && pkg install shc -y && pkg install binutils -y && termux-setup-storage`

Shizuku Permission : `cd /sdcard/ZmodXv3.0 && sh rish`

Setup : `sh setup.sh`

And Copy : `cd /data/local/tmp`

Open Menu : `./ZMODX`
